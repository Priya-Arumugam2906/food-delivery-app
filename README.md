Food Delivery App 

This is a responsive Food Delivery web application built using React JS. The project allows users to browse food items, add them to the cart, and place orders through a simple and interactive interface.

The goal of this project was to gain practical experience in building dynamic React applications and managing state efficiently.

Features

Browse food items by category

Add and remove items from the cart

View total price with delivery charges

Login / Signup popup UI

Order placement page

Responsive design

Smooth navigation using React Router

Tech Stack

React JS

JavaScript

HTML & CSS

React Router

Context API (for cart state management)

Run Locally

cd food-delivery-app
npm install
npm install react-router-dom
npm run dev
